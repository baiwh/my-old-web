HTML标签

1、标题：h1-h6。1大6小。
	浏览器会自动地在标题的前后添加空行。
          默认情况下，HTML 会自动地在块级元素前后添加一个额外的空行，比如段落、标题元素前后。
     <h1>...</h1>
2、水平线：
     <hr/>
3、注释：
     <!--...-->
4、段落：浏览器会自动地在段落的前后添加空行。（<p> 是块级元素）
     <p>..</p>
5、背景的颜色：
	主体背景颜色：<body style="background-color:yellow">
               ...
             </body>
	标题颜色：<h2 style="background-color:red">...</h2>
	段落颜色：<p style="background-color:green">...</p>
6、字体的颜色和尺寸：
	尺寸：<h1 style="font-family:verdana">...</h1>
	颜色：<p style="font-family:arial;color:read;fon-size:20px;">...</p>
7、文本对齐：
     <h1 style="text-align:center">...</h1>
8、文本格式化标签：<*>...</*>
	粗体：<b>
	大号字：<big>
	着重文字：<em>
	斜体字：<i>
	小号字：<small>
	加重语气：<strong>
	下标字：<sub>
	上标字：<sup>
	插入字：<ins>
	删除字：<del>
9、计算机输出标签： <*>...</*>   这些标签常用于显示计算机/编程代码。支持固定的字母尺寸和间距。
	计算机代码：<code>  不保留多余的空格和折行， 如需解决该问题，必须在 <pre> 元素中包围代码
	键盘码：<kbd>
	计算机代码样本：<samp>
	打字机代码：<tt>
	变量：<var>   定义数学变量
	预格式文本：<pre>...它保留了空格和换行。pre 标签很适合显示计算机代码
10、引用、引用和术语定义：<*>...</*>
	缩写：<abbr>...<abbr title="etcetera">etc.</abbr>..把鼠标移至缩略词语上时，
		title 可用于展示表达的完整版本， 对缩写进行标记能够为浏览器、翻译系统以及搜索引擎提供有用的信息。
	首字母缩写：<acronym>
	地址：<adress>  此元素通常以斜体显示。大多数浏览器会在此元素前后添加折行。
	文字方向：<bdo>
	长引用：<blockquote>   使用 blockquote 元素的话，浏览器会插入换行和外边距， 
		浏览器通常会对 blockquote 元素进行缩进处理。
	短引用：<q>  而 q 元素不会有任何特殊的呈现，浏览器通常会在 q 元素周围包围引号。
	引用、引证：<cite>   定义著作的标题。浏览器通常会以斜体显示 <cite> 元素。
	定义项目：<dfn>
11、链接：属性值本身就含有双引号，那么必须使用单引号
	指向网址：<a href="http://....">...</a>
	指向网页：<a href="...  .htm">...</a>
	用图像做链接：<a href="http://...">
	             <img src="...  .jpg" />
	             </a>
	在新窗口打开链接：<a href="http://..." target="_blank">...</a>
	跳转到指定位置（锚）：锚的名称可以是任何你喜欢的名字。可以使用 id 属性来替代 name 属性，命名锚同样有效。
	命名一个锚：<a name="tips">...</a>
	在同一个文档中创建指向该锚的链接：<a href="#tips">...</a>
	在其他页面中创建指向该锚的链接：<a href="http://.../html/html_links.asp#tips">...</a>
	跳出框架：<a href="/index.html" target="_top">...</a>
	发邮件：<a href="mailto:...@...?subject=Hello%20again">发送邮件</a>
               使用 %20 来替换单词之间的空格，这样浏览器就可以正确地显示文本了。
12、图像：插入动画图像的语法与插入普通图像的语法没有区别
          来自文件夹：<img src="...  .jpg" width="50" height="50" />
          来自网页：<img src="http://..." width="50" height="50" />
          替换文本：无法载入图像时，浏览器将显示这个替代性的文本而不是图像。
                    <img src="...  .jpg" alt="替换文本">
          背景图像：gif 和 jpg 文件均可用作 HTML 背景。如果图像小于页面，图像会进行重复。
                    <body background="... .jpg">
                    </body>
          文字中排列：bottom 对齐方式是默认的对齐方式。
                   下对齐：<img src="... .gif" align="bottom">
  中对齐：<img src="... .gif" align="middle">
  上对齐：<img src="... .gif" align="top">
          浮动：文本左侧：<img src="... .gif" align="left">
                文本右侧：<img src="... .gif" align="right">
          创建图像映射： 创建带有可供点击区域的图像地图。其中的每个区域都是一个超级链接。
			<img  src="...  .jpg"  border="0" usemap="#planetmap"  alt="Planets" />
		img 元素中的 "usemap" 属性引用 map 元素中的 "id" 或 "name" 属性（根据浏览器），
		所以我们同时向 map 元素添加了 "id" 和 "name" 属性。
		<map name="planetmap" id="planetmap">
		映射区域
		<area  shape="circle" coords="180,139,14"  
			href ="/example/html/venus.html"  target ="_blank"  alt="Venus" />
		</map>
		转化为图像映射：<a href="/example/html/html_ismap.html">
		   <img src="/i/eg_planets.jpg" ismap />
		   </a>
           定义图像地图：<map>
           定义图像地图中可点击区域：<area>
13、表格：每个表格由 table 标签开始。行（row）由 tr 开始。数据由 td 开始（table data）。
          数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等等。
一行一列：<table border="1">
          <tr>
               <td>1,1</td>
          </tr>
          </table>
一行三列：<table border="1">
          <tr>
               <td>1,1</td>
               <td>1,2</td>
               <td>1,3</td>
          </tr>
          </table>
两行三列：<table border="1">
          <tr>
               <td>1,1</td>
               <td>1,2</td>
               <td>1,3</td>
          </tr>
          <tr>
               <td>2,1</td>
               <td>2,2</td>
               <td>2,3</td>
          </tr>
          </table>
表头：<tr>
     <th>...</th>
     </tr>
空单元格： 加一个空格占位符，就可以将边框显示出来
                  <td> </td>
表格标题：<caption>...</caption>  标题默认居中
跨列（横向）：<tr>
             <th colspan="2">...</th>
             </tr>
跨行（纵向）：<tr>
             <th rowspan="2">...</th>
             </tr>
表格边距： 内容与其边框之间的空白：
          <table cellpadding="10" border="1" >... </table>  
           单元格之间的距离:
                 <table cellspacing="10" border="1" >... </table>
背景颜色：表格：<table bgcolor="red" border="1" >... </table>  
         单元格：<tr>
                    <td bgcolor="red">...</td>
                </tr>
背景图像：<table background="...  .gif" border="1" >... </table> 
   单元格：<tr>
              <td bgcolor="red">...</td>
          </tr>
单元格内排列对齐：<tr>
               <td align="left">1,1</td> 左对齐
               <td align="right">1,2</td> 右对齐
               <td align="conter">1,3</td> 居中
           </tr>
围绕表格的边框：四周：<table frame="box">...</table>
               上方：<table frame="above">...</table>
               下方：<table frame="below">...</table>
               上下：<table frame="hsides">...</table>
               左右：<table frame="vsides">...</table>       
14、列表： 列表项内部可以使用段落、换行符、图片、链接以及其他列表等等。
	无序列表：使用粗体圆点（典型的小黑圆圈）进行标记
	<ul>
	<li>...</li>
	</ul>
	有序列表： 使用数字进行标记。
	<ol>
	<li>...</li>
	</ol>
	指定开始序号：50起
	<ol start="50">
	<li>...</li>
	</ol>
	不同标记：<ol type="*">...</ol>  数字1，字母A，小写a。罗马大小写
	定义列表：<dl>
	<dt>...</dt>  名词
	<dd>...</dd>  解释
    </dl>
15、target属性：规定在何处打开
16、按钮：<button> 标签定义一个按钮。在 button 元素内部，可以放置内容，比如文本或图像。

HTML相关概念

1、块级元素<div>：在浏览器显示时，通常会以新行来开始（和结束）。
     内联元素<span>： 与 CSS 一同使用时，<span> 元素可用于为部分文本设置样式属性。
2、类：给元素的类定义CSS样式：
               <style>
               .class{...}
               </style>
            和块级元素连用：<div class="class">...</div>
            和内联元素连用：<h1>...<span class="class">...</span>...</h1>
3、响应式web：RWD 指的是响应式 Web 设计（Responsive Web Design），能够以可变尺寸传递网页，对于平板和移动设备是必需的
     自己创建：<html lang="en-us">...</html>
     现成的CSS框架：<head>
		<meta name="viewport" content="width=device-width, initial-scale=1">
		<link rel="stylesheet" 
			href="http://maxcdn.bootstrapcdn.com/bootstrap/3.2.0/css/bootstrap.min.css">
		</head>
4、框架：同一个浏览器窗口中显示不止一个页面。每份HTML文档称为一个框架，并且每个框架都独立于其他的框架。
          使用框架的坏处：开发人员必须同时跟踪更多的HTML文档，很难打印整张页面
          不能将 <body></body> 标签与 <frameset></frameset> 标签同时使用！
			不过，假如你添加包含一段文本的 <noframes> 标签，
			就必须将这段文字嵌套于 <body></body> 标签内。
	垂直框架：<frameset cols="25%,50%,25%">
	               <frame src="... .html">
	          ...
	               </frameset>
	水平框架：<frameset rows="...">.....</frameset>
	混合框架：嵌套一个<frameset>
	加noresize后框架不可移动：<frame src="... .html" noresize="noresize"/>
	导航框架：？
5、内联框架：<iframe src="url">...</iframe>  url：位置
	高度宽度：<iframe src="url" width="200" herght="200">...</iframe>
	删除边框：<iframe src="url" frameborder="0">...</iframe>
	作为链接的目标：<iframe src="url" name="a">...</iframe>
                         <p><a href="http://..." target="a">.....</a></p>
6、实体：文本中不能使用<和>。
       小于号（<或 <）和大于号（>或>）
       增加空格的数量，需要使用   或   字符实体
              还有&。"。'。
7、url：统一资源定位符，也被称为网址。URL 可以由单词组成，比如 “w3school.com.cn”，
		或者是因特网协议（IP）地址：192.168.1.253。
8、颜色：红绿蓝十六进制或RGB， 每种颜色的最小值是0（十六进制：#00）。最大值是255（十六进制：#FF）
		#000000=rgb(0,0,0) 。#FF0000=rgb(255,0,0)。#C0C0C0=rgb(192,192,192)
     颜色名： 仅仅有 16 种颜色名被 W3C 的 HTML4.0 标准所支持。它们是：aqua, black, blue, 
		fuchsia, gray, green, lime, maroon, navy, olive, 
		purple, red, silver, teal, white, yellow。
     安全色： 216 跨平台 web 安全色被用来确保：当计算机使用 256 色调色板时，所有的计算机能够正确地显示所有的颜色。
9、 文档类型：<!DOCTYPE> 声明帮助浏览器正确地显示网页。<!DOCTYPE> 不是 HTML 标签。
	它为浏览器提供一项信息（声明），即 HTML 是用什么版本编写的。

	......

	......
	<input type="submit" name="...">
    </form>

	<fieldset>

7、下拉元素：<form action="...">
	<select name="cars">

10、输入类型：<input type="..." >
	type="texe"   定义供文本输入的单行输入字段
	type="password"   密码字段，输入的字符会被做掩码处理（星号或实心圆）
	type="submit"   提交表单数据
	type="radio"   单选
	type="checkbox"   多选
	type="button"   按钮
	type="number"   数字值的输入  可以加限制：<input type="number" name="quantity" min="1" max="5">
	type="data"   日期  可限制： <input type="date" name="bday" max="1979-12-31">
	type="color"    颜色
	type="range"    滑块  能够使用如下属性来规定限制：min、max、step、value
	type="month"    选择月和年
	type="week"    选择周和年
	type="time"    选择时间（无时区）
	type="datatime"    选择时间和日期（有时区）
	type="datatime-local"    选择日期和时间（无时区）
	type="email"    电子邮件地址输入    某些智能手机会识别 email 类型，并在键盘增加 ".com" 以匹配电子邮件输入
	type="seatch"    搜索   表现类似常规文本字段
	type="tel"    电话号码
	type="url"    地址    某些智能手机识别 url 类型，并向键盘添加 ".com" 以匹配 url 输入
11、输入属性：
	初始值：value  <input type="text"  value="John">
	只读（不能修改）：readonly   <input type="text"  value="John"readonly>
	禁用：disabled   <input type="text"  value="John"disabled>
	输入框尺寸（以字符计）：size  <input type="text"  value="John" size="50">
	允许的最大长度：maxlength  <input type="text"  value="John" maxlength="10">
	自动完成：autocomplete  <form action="action_page.php" autocomplete="on">
	不对数据进行验证：nobalidate  <form action="action_page.php" novalidate>
	当页面加载时 <input> 元素应该自动获得焦点：autofocus   <input type="text" name="fname" autofocus>
	引用表单：form  <input type="text" name="lname" form="form1">
	高度和宽度：height width 仅用于 <input type="image"> 
	          <input type="image" src="img_submit.gif" alt="Submit" width="48" height="48">
	预定义：list   <input list="browsers" name="...">
	最大最小：max min   适用于如需输入类型：number、range、date、datetime、datetime-local、month、time 以及 week
	          <input type="date" name="bday" max="1979-12-31">
	允许用户输入一个以上的值： multiple   适用于以下输入类型：email 和 file。
	               <input type="file" name="img" multiple="multiple" />
	检查：pattern    适用于以下输入类型：text、search、url、tel、email、and password。
		使用全局的 title 属性对模式进行描述以帮助用户
	<input type="text" name="country_code" pattern="[A-z]{3}"
	title="三个字母的国家代码" />
	描述输入框预期提示： placeholder    适用于以下输入类型：text、search、url、tel、email 以及 password。
	<input type="search" name="user_search" placeholder="提示文字" />
	
	必填：required   适用于以下输入类型：text、search、url、tel、email、password、date pickers、
		number、checkbox、radio、and file.
	<input type="text" name="usr_name" required="required" />
	合法数字间隔：step    可与 max 以及 min 属性一同使用，来创建合法值的范围
	适用于以下输入类型：number、range、date、datetime、datetime-local、month、time 以及 week
	<input type="number" name="points" step="3" />

HTML5

1、视频：control 属性供添加播放、暂停和音量控件
	<video src="movie.ogg"  controls="controls">...</video>
	加尺寸：<video width="320" height="240" src="..." controls="controls">...</video>
	不同格式的: ideo 元素允许多个 source 元素。source 元素可以链接不同的视频文件。浏览器将使用第一个可识别的格式
	<video width="320" height="240" controls="controls">
	<source src="....mp4" type="video/mp4>
	<source src="....ogg" type="video/ogg>
	<source src="....webm" type="video/webm>
	</video>
2、<video>标签属性：autoplay就绪后马上播放
	controls向用户显示控件，如播放按钮
	loop当媒介文件完成播放后再次开始播放
	preload页面加载时加载视频并预备播放（如果用outoplay则忽略）
	src要播放视频的URL
3、音频：control 属性供添加播放、暂停和音量控件。<audio> 与 </audio> 之间插入的内容是供不支持 audio 元素的浏览器显示的
	<audio src="....ogg" controls="controls"></audio>
4、拖放：使元素可拖动：<img draggable="true" />
	拖动什么。放到何处。进行放置。
5、画布：JS
6、SVG：指可伸缩矢量图形 (Scalable Vector Graphics)，用于定义用于网络的基于矢量的图形，
	使用 XML 格式定义图形，图像在放大或改变尺寸的情况下其图形质量不会有损失XML

XHTML

1、id 属性代替 name 属性：HTML 4.01 针对下列元素定义 name 属性：a, applet, frame, iframe, img, 和map。
	在 XHTML 中不鼓励使用 name 属性，应该使用 id 取而代之。
	在 "/" 符号前添加一个额外的空格，以使 XHTML 与当今的浏览器相兼容
2、语言属性：<div lang="no" xml:lang="no">...</div>
3、DTD：文档声明类型。<!DOCTYPE htmlPUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
		"http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
	STRICT（严格类型）： 包含没有被反对使用的或不出现在框架结构中的元素和属性
		<!DOCTYPE .../xhtml1-strict.dtd">
	TRANSITIONAL（过渡类型）：包含严格 DTD 中的一切，外加那些不赞成使用的元素和属性
		<!DOCTYPE .../xhtml1-transitional.dtd">
	FRAMESET（框架类型）： 包含过渡 DTD 中的一切，外加框架。
		<!DOCTYPE .../xhtml1-frameset.dtd">
