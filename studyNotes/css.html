CSS基础知识

1、基础语法：HTML选择器{属性：值；}属性间用分号隔开
	除了英文单词 red，还可以使用十六进制的颜色值 #ff0000： p { color: #ff0000; }
	可以使用 CSS 的缩写形式： p { color: #f00; }
	可以通过两种方法使用 RGB 值：当使用 RGB 百分比时，即使当值为 0 时也要写百分比符号。
	但是在其他的情况下就不需要这么做了。比如说，当尺寸为 0 像素时，0 之后不需要使用 px 单位，因为 0 就是 0，无论单位是什么。
	p { color: rgb(255,0,0); }
	p { color: rgb(100%,0%,0%); }
	如果值为若干单词，则要给值加引号
	CSS对大小写不敏感，class 和 id 名称对大小写是敏感的
2、高级语法： 选择器进行分组，被分组的选择器就可以分享相同的声明。选择器间用逗号隔开
	子元素将继承最高级元素所拥有的属性
	不希望子元素继承，可以针对不想继承的子元素单独创建一个特殊规则
3、派生选择器：li strong{...}
4、ID选择器（只能用一次）：为标有特定 id 的 HTML 元素指定特定的样式。以 "#" 来定义。
    常常用于建立派生选择器。也可以独立发挥作用
5、类选择器（可以用多次）：以一个点号来定义，以class="类"来引用。
	类名的第一个字符不能使用数字
	定义一个类：.center {text-align: center}
	引用一个类：<h1 class="center">
	用作派生选择器：.fancy td{...}
	基于他们的类而被选择：td.fancy{...}
	可以将类 fancy 分配给任何一个表格元素任意多的次数
6、属性选择器：可以为拥有指定属性的 HTML 元素设置样式，而不仅限于 class 和 id 属性
	属性选择器：[title]{color:red;}
	属性和值选择器：[title=hello]{color:red;}
	属性和多值选择器：为包含指定值的 title 属性的所有元素设置样式。适用于由空格分隔的属性值，如：hello world
	[title~=hello]{color:red;}
	为带有包含指定值的 lang 属性的所有元素设置样式。适用于由连字符分隔的属性值，如：hello-world
	[lang|=hello]{color:red;}
	表单样式：input[type="..."]{...}
7、创建插入样式表：
	外部：样式需要应用于很多页面时，外部样式表将是理想的选择
	可以通过改变一个文件来改变整个站点的外观。
	每个页面使用 <link> 标签链接到样式表。<link> 标签在（文档的）头部
	浏览器会从文件 mystyle.css 中读到样式声明，并根据它来格式文档
	<head>
	<link rel="stylesheet" type="text/css" href="mystyle.css" />
	</head>
	外部样式表可以在任何文本编辑器中进行编辑。文件不能包含任何的 html 标签。样式表应该以 .css 扩展名进行保存
	hr{...}
	p{...}
	body{...}
	内部： 单个文档需要特殊的样式时，可以使用 <style> 标签在文档头部定义内部样式表
	<head>
	<style type="text/css">
	  hr {color: sienna;}
	  p {margin-left: 20px;}
	  body {background-image: url("images/back40.gif");}
	</style>
	</head>
	内联： 当样式仅需要在一个元素上应用一次时
	<p style="color: sienna; margin-left: 20px">
	...
	</p>
	多重样式表： 如果某些属性在不同的样式表中被同样的选择器定义，那么属性值将从更具体的样式表中被继承过来，
	例如：外部样式表比内部更具体，内部没有的属性将被继承于外部样式表，而内部有的会被内部样式表中的规则取代
8、注释：/*....*/

CSS样式：背景(纯色或图像)

1、颜色：background-color 不能继承，其默认值是 transparent（透明）
     p{baceground-color:red;}
2、图像：所有背景属性都不能继承
     p{background-imge:url(图片地址.gif);}
3、重复：页面上对背景图像进行平铺，属性值 repeat 在水平垂直方向上都平铺。
	repeat-x 和 repeat-y 只在水平或垂直方向上重复，no-repeat 则不允许图像在任何方向上平铺。
	body{
		background-image:url(....gif);
		background-repeat:repeat-x;
	}
4、定位：top、bottom、left、right。还可以使用长度值，如 100px 或 5cm，百分数值。
           关键字： 位置关键字可以按任何顺序出现，只要保证不超过两个关键字 - 一个对应水平方向，另一个对应垂直方向。
           如果只出现一个关键字，则认为另一个关键字是 center。
		body{
			background-image:url(....gif);
			background-repeat:no-repeat;
			background-position:center;
		}
	百分数：百分数值同时应用于元素和图像。如果只提供一个百分数值，所提供的这个值将用作水平值，垂直值将假设为 50%。
        body{
			background-image:url(....gif);
			background-repeat:no-repeat;
			background-position:50% 50%;
		}
	长度：元素内边距区左上角的偏移。偏移点是图像的左上角。
		左上角将在元素内边距区左上角向右 50 像素、向下 100 像素的位置上
        body{
			background-image:url(....gif);
			background-repeat:no-repeat;
			background-position:50px 100px;
		}
5、关联：文档比较长，那么当文档向下滚动时，背景图像也会随之滚动。当文档滚动到超过图像的位置时，图像就会消失。
	这个属性可以防止这种滚动，声明图像相对于可视区是固定的（fixed），因此不会受到滚动的影响
    body{
		background-image:url(....gif);
		background-repeat:no-repeat;
		background-attachment:fixed;
	}
6、所有背景属性放在一个声明中：
	body{ 
		background: #ff0000 url(/i/eg_bg_03.gif) no-repeat fixed center; 
	}

CSS样式：文本

1、文本颜色：<style type="text/css">
     		h1{color:#005533}
		</style>
		<h1>...</h1>
    文本背景颜色：h1{backgro-indent:5em;}
    悬挂缩进： 对一个段落设置了负值，那么首行的某些文本可能会超出浏览器窗口的左边界。
			为了避免出现这种显示问题，建议针对负缩进再设置一个外边距或一些内边距
          p{text-indent:-5em;padding-left:5em}
     百分比：百分数要相对于缩进元素父元素的宽度
		<style type="text/css">
			div {width: 500px;}
			p {text-indent: 20%;}
		</style>
		<div>
	    		<p>this is a paragragh</p>
		</div>
     继承：text-indent 属性可以继承
3、水平对齐： 将块级元素或表元素居中，要通过在这些元素上适当地设置左、右外边距来实现。
	左对齐：hi{text-align:left}
	右对齐：hi{text-align:right}
	居中：hi{text-align:center}
	两端对齐：justify
	从父元素继承：inherit
4、字间隔：p{word-spacing:30px;}
5、字母间隔：p{letter-spacing:30px;}
6、字符转换：处理文本大小写
	不做任何改动，使用源文档原有的：p{text-transform:none;}
	全大写：p{text-transform:uppercase;}
	全小写：p{text-transform:lowercase;}
	只对每个单词的首字母大写：p{text-transform:capitalize;}
7、文本装饰：会替换不会积累
	下划线：h1{text-decoration:underline;}
	顶端上划线：h1{text-decoration:overline;}
	中间贯穿线：h1{text-decoration:line-through;}
	闪烁：h1{text-decoration:blink;}
	同时两个：h1{text-decoration:underline overline;}
	不做任何改动，使用源文档原有的：p{text-decoration:none;}
8、空白符：各个字之间只会显示一个空格，同时忽略元素中的换行：
		p{white-space:normal;}
	当 white-space 属性设置为 pre 时，浏览器不会合并空白符，也不会忽略换行符
		p{white-space:pre;}
	防止元素中的文本换行，除非使用了一个 br 元素
    	p{white-space:nowrap;}
	保留空白符序列，但是文本行会正常地换行
    	p{white-space:pre-wrap;}
	合并空白符序列，但保留换行符
    	p{white-space:pre-line;}
9、行间距：
	用百分比：p{line-height:90%}
	用像素值：p{line-height:10px}
	用数值：p{line-height:0.9}

CSS样式：字体

1、指定字体：
	通用字体：Serif 字体。Sans-serif 字体。Monospace 字体。Cursive 字体。Fantasy 字体
	     body{font-family:sans-serif;}
	指定字体：如果用户代理上没有安装想要的字体，就只能使用默认字体。可以通过结合特定字体名和通用字体系列来解决这个问题
	     body{font-family:某字体,sans-serif;}
2、字体风格：normal - 文本正常显示.italic - 文本斜体显示.oblique - 文本倾斜显示
          p{font-style:normal}
3、字体变形：小型大写字母
          p{font-variant:small-caps;}
4、字体加粗： 100 ~ 900 为字体指定了 9 级加粗度。如果一个字体内置了这些加粗级别，那么这些数字就直接映射到预定义的级别，
		100 对应最细的字体变形，900 对应最粗的字体变形。数字 400 等价于 normal，而 700 等价于 bold。 
		如果将元素的加粗设置为 bolder，浏览器会设置比所继承值更粗的一个字体加粗。
		与此相反，关键词 lighter 会导致浏览器将加粗度下移而不是上移。
                    p {font-weight: 900}
5、字体大小：值可以是绝对或相对值如果没有规定字体大小，普通文本（比如段落）的默认大小是 16 像素 (16px=1em)。
	绝对值：将文本设置为指定的大小，不允许用户在所有浏览器中改变文本大小（不利于可用性），
		绝对大小在确定了输出的物理尺寸时很有用
	相对大小：相对于周围的元素来设置大小，允许用户在浏览器改变文本大小
	用像素：p{font-size:50px;}
	用em： 1em 等于当前的字体尺寸。 px/16=em
	     p{font-size:1.5em;}
	结合使用百分比和em： 在所有浏览器中均有效的方案是为 body 元素（父元素）以百分比设置默认的 font-size 值
		body{font-size:100%}
		p{font-size:1.5em;}
6、所有字体在一个声明内：
	<style type="text/css">
		p.ex1 { font:italic arial,sans-serif; }
		p.ex2 { font:italic bold 12px/30px arial,sans-serif; }
	</style>

CSS样式：链接

1、设置链接样式：为了使定义生效，a:hover 必须位于 a:link 和 a:visited 之后，a:active 必须位于 a:hover 之后。
	“a”可以改为其他，如img
	<head>
		<style>
			a:link {color:#FF0000;}    /* 未被访问的链接 */
			a:visited {color:#00FF00;} /* 已被访问的链接 */
			a:hover {color:#FF00FF;}   /* 鼠标指针移动到链接上 */
			a:active {color:#0000FF;}  /* 正在被点击的链接 */
		</style>
	</head>
	<body>
		<p><b><a href="/index.html" target="_blank"
			>这是一个链接</a></b></p>
		<p><b>注释：</b>！！</p>
		<p><b>注释：</b>为了使定义生效，！！</p>
	</body>
2、文本修饰： 用于去掉链接中的下划线
	a:link {text-decoration:none;}    /* 未被访问的链接 */
	a:visited {text-decoration:none;} /* 已被访问的链接 */
	a:hover {text-decoration:underline;}   /* 鼠标指针移动到链接上 */
	a:active {text-decoration:underline;}  /* 正在被点击的链接 */
3、背景颜色：a:link {background-color:#B2FF99;}
4、改变尺寸：a.two:hover {font-size:150%;}
               <a class="two" href="...">...</a>
5、创建链接框：
	<head>
		<style>
			a:link,a:visited
			{
			display:block;
			font-weight:bold;
			font-size:14px;
			font-family:Verdana, Arial, Helvetica, sans-serif;
			color:#FFFFFF;
			background-color:#98bf21;
			width:120px;
			text-align:center;
			padding:4px;
			text-decoration:none; }
			a:hover,a:active { background-color:#7A991A; }
		</style>
	</head>
	<body>
	     <a href="/index.html" target="_blank">W3School</a>
	</body>

框模型

1、内外边距：*{margin:0;padding:0;}
2、内容区域的宽和高：*{width:0;herght:0;}
3、概述：
	element : 元素。
	padding : 内边距，也有资料将其翻译为填充。
	border : 边框。
	margin : 外边距，也有资料将其翻译为空白或空白边。
4、内边距：padding 属性接受长度值或百分比值，但不允许使用负值
	各边都有 10 像素的内边距：p{padding:10px;}
	上、右、下、左：p{padding:10px 0.25em 2ex 20%;}
	单边页边距：padding-top/right/bottom/left
	内边距的百分比：百分数值是相对于其父元素的 width 计算的
5、边框样式：groove凹槽/ridge垄状/inset凹/outset凸/none
	简写样式（上开始顺时针）：p{border-style:solid dotted dashed double;}
	单边样式（单边在简写后）：p{border-style:solid;border-top/right/bottom/left-style:none;}
6、边框宽度：1px/0.1em/thin/medium/thick/none，一定先使用border-style
	简写：p{border-style:solid;border-width:15px 0.1em thick 1px;}
	值复制简写（上下 左右）：p{border-style:solid;border-width:15px 0.1em 15px 0.1em;}，等价于
	        p{border-style:solid;border-width:15px 0.1em;}
	单边：p{border-style:solid;border-top/right/bottom/left-width:15px;}
7、边框颜色：透明色transparent
	 简写：p{border-style:solid;border-color:blue rgb(..,..,..) #...... red;}
	复制值简写：p{border-style:solid;border-color:blue red;}
	单边：p{border-style:solid;border-top/right/bottom/left-color:red;}
8、属性写一起：
	四边属性：p{border:thin double rgb(..,..,..)}
	某边属性：p{border-top:thin double rgb(..,..,..)}
9、外边距：auto/in/px/cm/em/%
     p{margin:...;}
10、单边外边距：margin-top/right/bottom/left
11、外边距合并： 只有普通文档流中块框的垂直外边距才会发生外边距合并。行内框、浮动框或绝对定位之间的外边距不会合并。

定位

1、相对定位：相对于正常位置，如果将 top 设置为 20px，那么框将在原位置顶部下面 20 像素的地方。
	如果 left 设置为 30 像素，那么会在元素左边创建 30 像素的空间，也就是将元素向右移动。元素仍然占据原来的空间
	p{position:relative;left:20px}
	p{position:relative;left:-20px}
2、绝对定位：元素框从文档流完全删除。 绝对定位的元素的位置相对于最近的已定位祖先元素，如果元素没有已定位的祖先元素，
	那么它的位置相对于最初的包含块。可以通过z-index来控制堆放次序
     p{position:absolute;left:20px;top:15px}
3、固定定位：其包含块是视窗本身
     p{position:fixed;left:20px;top:50px}
4、可以使用固定值，百分比
5、溢出元素滚动条：overflow 属性可以确定是否显示滚动条，如果值为scroll，不论是否需要，用户代理都会提供一种滚动机制
     p{width:150px;height:150px;overflow:scroll}
6、隐藏溢出元素：
     p{width:150px;height:150px;overflow:hidden}
7、浏览器自动处理溢出：
     p{width:150px;height:150px;overflow:auto}
8、设置元素形状：（剪切）
     img{position:absolute; clip:rect(0px 50px 200px 0px)}
9、垂直排列图像：
     顶部对齐：img{vertical-align:text-top}
     底部对齐：img{vertical-align:text-buttom}
10、将一个元素置于另一个之后：默认的 z-index 是 0。Z-index -1 拥有更低的优先级
     img{position:absolute;z-index:-1}
11、将一个元素置于另一个之前：
     img{position:absolute;z-index:1}
12、浮动：img{float:right}
13、带标题的图像浮动：
	<html>
	<head>
	<style type="text/css">
		div{
		float:right;
		width:129px;
		margin:0 0 15px 20px;
		padding:15px;
		border:1px solid black;
		text-align:center;
		}
	</style>
	</head>
	<body>
	<div>
		<img src="/c/....gif"/></br>
		标题
	</div>
	</body>
	</html>
14、段落首字母浮动于左侧：用<span>X</span>
15、清除元素的侧面：侧面不允许出现浮动元素
     如果声明为左边或右边清除，会使元素的上外边框边界刚好在该边上浮动元素的下外边距边界之下。
     img{float:left;clear:both;}

选择器

1、选择器分组：
     h1,h2,p,table{color:gray;}
2、通配符选择器：文档中所有元素的一个分组选择器
     *{color:gray;}
3、声明分组：一个分组写多个属性
     p{color:gray;background:black;}
4、类选择器：
     *.class1{color:gray;}或者.class1{color:gray;}
     <p class="class1">....</p>
5、类+元素选择器：选择器现在会匹配 class 属性包含 important 的所有 p 元素，但是其他任何类型的元素都不匹配， 
	其 class 属性值为class的所有段落
     p.class1{color:gray;}
6、多类选择器：同时包含class1和2的元素，将带有.class1和.class2和.class1.class2的样式。多类选择器顺序不限
     .class1{color:gray;}
     .class2{background:black;}
     .class1.class2{font-weight:bold;}
     <p class="class1 class2">.....</p>
7、ID选择器：只能在文档中使用一次，区分大小写
     *#id1{color:gray;}或者#id1{color:gray;}
     <p id="id1">....</p>
8、属性选择器：对包含某属性的所有元素
     *[title]{color:gray;}
9、多属性选择器：
     *[title][href]{color:gray;}
10、具体属性值选择器：必须完全匹配
     a[href="http://baidu.com"]{color:gray;}
     a[href="http://baidu.com"][title="baidu"]{color:gray;}
11、部分属性值选择器：可应用于任何属性，要加空格
     p[class~="class1"]{color:gray;}
     <p class="class1 class2">......</p>
     img[title~="t"]{color:gray;}
     <img title="t tttt"/>
12、子串匹配属性选择器：不加空格
     [abc^="def"]选择abc属性以def开头的所有元素
     [abc$="def"]选择abc属性以def结尾的所有元素
     [abc*="def"]选择abc属性包含子串def的所有元素
13、特定属性选择器：以en或en-开头的
     *[lang|="en"]{color:gray;}
     <p lang="en">.....</p>
     <p lang="en-enen">.....</p>
14、后代选择器：只对p元素中的em应用。不论 em 的嵌套层次多深，都能应用
     p em{color:gray;}
     <p>........<em>,,,</em>....</p>
15、子元素选择器：不希望选择任意的后代元素，而是希望缩小范围，只选择某个元素的子元素。选择只作为 h1 元素子元素的 strong 元素
     h1>strong{color:red;}
     <h1>...<strong>,,,</strong>...</h1>
16、相邻兄弟选择器：选择紧接在另一个元素后的元素，而且二者有相同的父元素。增加紧接在 h1 元素后出现的段落的上边距
     h1+p{margin-top:50px;}
17、伪类语法：
     选择器：伪类{属性：值}
     input：focus{background-color:gray;}
     类+伪类：input.class1：focus{background-color:gray;}
18、首个子对象：仅第一个匹配的元素
     p:first-chile{font-werght:bold;}
     p>i:first-child{...}所有<p>元素中的第一个<i>元素
     p:first-child i{...}所有作为第一个子元素的<p>元素中的所有<i>元素
19、伪元素语法：
     选择器：伪元素{属性：值；}
20、first-line伪元素：向文本的首行设置特殊样式
     p:first-line{...}
21、first-letter伪元素：向文本的首字母设置特殊样式
     p:first-letter{...}
22、伪元素+类：
     p.class1:first-line{...}
23、before伪元素：在元素的内容前面插入新内容
     h1:before{content:url(/c/....gif)}
24、after伪元素：
     h1:after{content:url(/c/....gif)}
