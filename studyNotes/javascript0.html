第一部分。JavaScript基础

1.对象表示法
	用句点方式。可以扩展任意多次。
	对象1.对象2.对象3
	对象1.对象2.属性或方法
2.模态对话框（弹出）
     alert（“内容”）；
3.向html文档写入
     document.write（“内容”）；
4.读取document对象的属性
     alert（document.title）；
5.添加JavaScript
     <script>...</script>
	<script src='....js'></script>
6.注释
     //.....或者/*....*/
7.变量
     var price=8;
     vat name="John";
8.操作符
     算数+-*/：var sum=4+3;
     用变量名称+-*/：var first=1;
               var second=2;
               var sum=first+second;
     求余数：%
     自增自减++ --：items++；
     加等+-*/：total+=5;等价于total=total+5;
9.字符串操作
     var firstname="john";
     var surname="doe";
     var fullname=firstname+" "+surname;
     字符串+数值=字符串（数值会被转换成字符串）
10.onclick事件处理器
     直接给HTML标签添加属性。
     <input type ="button" onclick="JavaScript语句" value="click me"/>
11.onmouseover事件处理器
     鼠标指针进入某元素区域时。

     和onmouseover组合可以实现鼠标进入和离开某区域时的图像变化。

13.创建函数

14.调用函数
     函数名称（）；
     <HTML标签 onclick="函数名称（）" />
     放到<head>区域，可以从页面的任何位置调用函数
15.给函数传递参数
     function cube(x)

16.多参数
     function cube(a,b)
     {alert(a*b);}

17.返回值

18.变量作用域
     函数内部定义的变量制作用于函数的内部，局部变量。函数外的为全局变量。

第二部分。编程

1.alert
     弹出对话框，只有消息和按钮。把字符串作为参数，没有返回值
2.confirm
     弹出对话框，可选择按钮。有返回值。ok返回真，cancel返回假。
3.prompt
     允许输入信息的模态对话框。可选择按钮。有返回值。输入内容，按ok返回输入的值。没输入内容，返回默认值。cancel返回null。
     prompt("问题","答案,默认值");
4.根据ID选择元素
     从HTML页面里选择具有某个特定ID的元素，秩序把相应元素的ID作为参数来调用document对象的getElementById（）方法。
		会返回特定的ID的页面元素所对应的DOM对象
     <div id="div1">     ...     </div>
     var mydiv=document.getElementById("div1");
     读取或设置特定页面元素内部的HTML内容
     访问/读取：var myDiv=document.getElementById("div1").innerHTML;
     设置/删除/改变/替换：document.getElementById("div1").innerHTML="新内容";
5.访问浏览器历史记录：
     history只有一个属性，即它的长度，表示访问过的页面的数量
     前进：history.forward();
     后退：history.backward();
     跳转到历史记录的某页，参数是正或负的整数：history.go(-3);
     找到历史列表里第一个匹配的URL，参数为字符串：history.go("匹配内容");
6.location对象
     当前加载页面的URL信息。协议//主机名：端口/路径 搜索 hash
7.导航至新页面：
     直接设置对象的href属性，原始页面还保留在浏览器的历史记录里，back按钮可返回以前页面：location.href='www.newpage.com';
     直接替换当前页面，从历史记录列表里删除，浏览器和历史记录都会用新的URL来代替旧的：location.replace('www.newpage.com');
8.刷新页面：
     重新加载当前页面，如果浏览器保存了当前页面，就会加载缓存内容：location.reload();
     从服务器获得页面数据：document.reload(true);
9.navigator对象：
     浏览器本身的数据，用来检测浏览器功能
10.日期
     当前日期：var mydate=new Date();
     获取当前日期：var xxx=mydate.getxxx();附录
     指定日期字符串：var date=new Date("October 22,1995 10:00:00")
     指定日期参数：var date2=new Date(年，月，日，时，分，秒，毫秒）
               前三个参数必填，后边的自选，后边的未指定的用0代替
11.设置和编辑日期
     当前日期时间：var date=new Date();
     更改成某日：date.setDate(某日);
     日期增加n天：date.setDate(date.getDate()+n);
12.math对象
     var num=11.5
     向上取整：alert(Math.ceil(num));
     向下取整：alert(Math.floor(num));
     四舍五入：alert(Math.round(num));
13.一组数中的最值（max/min）
     document.write(Math.max(n1,n2,n3));
14.随机数
     function myRand(range){
          retrn Math.round(Math.random()*range);}
     先用Math生成随机数，乘变量range来扩大范围，再利用round（）去除小数
     或者直接调用myRand（100），得到0-100的随机数
15.数学常数
     在计算中直接用，如PI圆周率：var area=Math.PI*radius*radius;
16.关键字with
     代码块里的语句在调用特定对象的方法时可以不必指定这个对象
     with(Math){
          var myRand=random();
          var big=max(3,4,5);}
17.非数值（NaN）
     非数值的数据当做数值处理，返回值就是NaN,利用isNaN（）可以检测非数值
     isNaN(3);  //false
     isNaN("three");  //trrue
18.把字符串强制转换为数值格式
     解析一个字符串并返回一个浮点数（首字符是数字会一直解析到数字结束，不是数字的返回NaN）：parseFloat("字符串");
     同上，但返回一个整数，可以有第二个可选参数，用于指定数值的基，返回二/八/十...进制：parseInt("字符串");
19.无穷大（±2^53）
     判断数值是否无穷大，把参数转换为数值，结果是NaN、正无穷大（Infinity）、负无穷大（-Infinity），
		返回false，其他情况返回ture：isFinite(函数);
20.字符串
     用单引号或者双引号定义：var myString="a string";
     定义空字符串：var mystring="";
     不能在字符串中出现的特殊字符用反斜线\
21.字符串方法
     连接字符串：var string=string1.concat(string2);
     返回子字符串第一次出现的位置（找不到返回-1）：string.indexOf(‘子字符串’);
     同上，返回最后一次的位置：string.lastIndexOf(‘子字符串’);
     找到子字符串并替换：var string1=string.replace("子字符串","替换的");
     把字符串分解为多个子字符串的数组并返回新数组：var newArray=string.split(" ");
     从指定的索引位置开始提取字符并返回，提取字符数量可选：var sub=string.subsre(指定位置,字符数量);
     全部转换大小写：var sub=string.toLowerCase();    或者toUpperCase
22.布尔值
     对布尔类型的变量赋值时注意不要加引号。true自动转换为1，false为0。
     类假：布尔false、未定义undefined、null、0、NaN、空字符串“”
     类真：非0
23.数组
     可以保存任何数据类型
     创建数组：var myArray=new Array();或者var myArray=[];
     数组初始化：var myArray=new Array['a1','a2','a3'];
               或者var myArray=[];
                    myArray[0]='a1';
                    myArray[1]='a2';
     数组长度：myArray.length
24.数组方法
     合并多个数组：var array=array1.concat(array2);
     把数组元素连接形成字符串：var array=myArray.join();
                              加分隔符：var array=myArray.join("分隔符");
     同上，用逗号分隔：var array=myArray.toString();
     找到指定元素在数组里第一次出现的位置并返回索引值，未找到返回-1：myArray.indexOf('指定元素');
     同上，最后一次出现的位置：myArray.lastIndexOf();
     从当前数组提取一个子集：var son=myArray.slice(开始的索引值,要提取的元素的数量);
     把数组按字母顺序排列：myArray.sort();
     添加或删除指定的一个或多个元素，会改变原数组，返回值是被删元素：array.splice(操作的位置,删除元素个数0表示不删除,
		[要插入的新元素列表，可选]);
25.if
     if（条件为真） 执行操作
     if(true) message="";
     非：！
     if(a<12) message="";
     比较操作符：等于==、值和类型都相等===、不等于！=、     
     if(...)
          {...;}
     else if(...)
          {...;}
     else{...;}
	简便语法：（条件为真）？[条件为真执行的]:[条件为假执行的];
26.switch
     switch(要判断的变量){
     case "...":
          message="满足当前条件时执行的语句";
          break;
     defaule:
          任何case都不满足时执行的操作；}
27.逻辑操作符
     与&&。或||
28.while
     只要条件判断结果为true，就会反复执行。不满足条件，循环会停止，执行花括号后的语句
     while(条件)
          {...;}
29.do...while
     条件判断前，代码块会先执行一次
     do{...;}
     while(...)
30.for
     for(初始条件；判断条件；循环后修改技术变量）
          {执行；}
     如果没有提前声明技术变量，可以在for语句初始条件里用var声明
31.for...in
     用于在对象的属性里进行循环，不必考虑使用循环计数器，或是判断循环结束的条件。
		循环会对集合中的每个对象执行一次，然后结束。可以操作任何对象的属性。
     for(...in...){...;}
32.定时器
      让程序延迟特定一段时间再执行
33.setTimeout()
      第二个参数所指定的毫秒之后，调用第一个参数。
      <div id="id1">...</div>
      function hide(elementId)
          {document.getElementById(elementId).style.display+'none';}
     window.onload=function()
          {setTimeout("hide('id1')",3000);}
     会返回一个值，想稍后取消定时器函数，通过将返回值传入给clearTimeout()来引用
     var timer1=setTimeout("hide('id')",3000);
     clearTimeout(timer1);
34.setInterval()
     同上，但是会重复执行，在两次执行之间，延迟第二个参数所指定的毫秒数。

	在函数里使用this，指向函数的父对象

          alert(this.info);}
7.使用构造函数
     创建可能具有多个实例的对象，创造某种模板，方便实现多次实例化
     先声明一个函数，然后在函数里使用this
     function myObjectType(){
          this.info="...";
          this.showInfo=function(){
               alert(this.info);}}

     var myNewObject=new myObjectType();
9.构造函数参数
     给构造函数传递一个或多个参数来定制对象
     function Person(personName){
          this.name=personName;
          this.showInfo=function(){
               alert(this.info);}}
     var person1=new Person('a');
     var person2=new Person('b');
     多个参数function Person(personName，time){...}
               var person1=new Person('a','2017');

          内容;}
11.继承
     先设计出通用的对象类型，再继承来不断细化得到更特定的类型
     function a(){...}
     function b(){...}
     b.prototype=new a();
12.扩展内置对象
